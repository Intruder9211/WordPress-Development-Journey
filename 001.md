--> As i Joined as a wordpress Developer intern there are not much about coding during my internship
    journey but as a developer mindset how i can be alive without code 
    So in a section where i used image carousel widget for animation there are only timed animation present
    like after 2-3 second another image shown.
    But i wanted to a loop animation where images continously moving so i used custom css in wordpress.
    Elementor’s Image Carousel doesn’t natively give you that smooth, infinite left-moving ticker feel. 
    What i'm trying to build is closer to a marquee-style continuous loop, not a standard slider. 
    Go to:
    Advanced → CSS Classes
    Add:
    infinite-scroll-carousel
    Add this Custom Css:
    .infinite-scroll-carousel .swiper-wrapper {
  transition-timing-function: linear !important;
}

.infinite-scroll-carousel .swiper-wrapper {
  animation: scrollLeft 20s linear infinite;
}

@keyframes scrollLeft {
  0% {
    transform: translateX(0);
  }
  100% {
    transform: translateX(-50%);
  }
}

/* Prevent interaction pause */
.infinite-scroll-carousel .swiper-wrapper {
  pointer-events: none;
}
